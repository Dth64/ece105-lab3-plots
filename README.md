# ECE105 Lab 3 - Sensor Data Visualization

## Project Description
This project generates synthetic temperature sensor data using NumPy and visualizes it using Matplotlib. The script creates scatter plots, histograms, and box plots.

## Installation
Install required packages:
pip install numpy matplotlib

## Usage
Run the script:
python generate_plots.py

## Output
The script generates:
- Scatter plot
- Histogram
- Box plot

Saved file:
sensor_analysis.png
